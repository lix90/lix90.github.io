pip 常用命令

搜索模块
pip search <pkg_name>

安装模块
pip install <pkg_name>
pip install <pkg_name>==<version>

升级模块（如果不提供版本号，升级到最新版本）
pip install --upgrade <pkg_name>>=<version>

卸载模块
pip uninstall <pkg_name>

显示当前环境中模块的清单
pip freeze
重定向到文本文件
pip freeze > req.txt
使用清单
pip install -r req.txt
